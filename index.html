Linearity Test

рҹ’Ў Golden Rule: T[aВ·xвӮҒ + bВ·xвӮӮ] = aВ·T[xвӮҒ] + bВ·T[xвӮӮ]

Disqualifiers: cos(В·), sin(В·), ln(В·), e^x, xВІ, |x|, +c

Time-Invariance Test

  1. Shift input: x(t) вҶ’ x(tвҲ’tвӮҖ) вҶ’ get yвӮҒ(t)
  2. Shift output: t вҶ’ (tвҲ’tвӮҖ) in y(t) вҶ’ get y(tвҲ’tвӮҖ)
  3. If yвӮҒ(t) = y(tвҲ’tвӮҖ) вҶ’ TIME INVARIANT вң“

Causality

Causal: Output depends on present & past inputs only. (e.g., y[n] = x[n] + x[n-1])

Non-Causal: Needs future inputs. (e.g., y[n] = x[n+2] вҲ’ x[n])
