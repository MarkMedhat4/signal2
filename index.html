🔄 Even & Odd Decomposition

Even: xₑ(t) = [x(t) + x(−t)] / 2
Odd: xₒ(t) = [x(t) − x(−t)] / 2
Even × Even = Even
Odd × Odd = Even
Odd × Even = Odd
∫ Odd dt over [−T,T] = 0

📅 Periodicity

CT:x(t+T₀) = x(t)
DT:x[n+N] = x[n]
Composite: T₁/T₂ rational → Periodic, T₀ = LCM(T₁,T₂)
T₁/T₂ irrational → NOT Periodic

📏 Linearity Test

1Apply x₁ → get y₁, apply x₂ → get y₂
2Apply (ax₁ + bx₂) → get y₃
3Check: y₃ = ay₁ + by₂ ?
Disqualifiers (auto NL):
cos(·) sin(·) ln(·) e^x x² |x| +constant

⏱️ Time Invariance Test

1Shift input: x(t) → x(t−t₀) → compute y₁(t)
2Take output y(t) → replace t with (t−t₀) → y₂(t)
3If y₁ = y₂ → TIME INVARIANT ✅
TV Clues: coefficient depends on t (like eᵗ, t², sin(t)·x(t)), or input has time scaling like x(at)

⏩ Causality

Causal: Output depends ONLY on present & past inputs

y[n] = x[n] + x[n−1] ✅ Causal

Non-Causal: Needs future inputs

y[n] = x[n+2] − x[n] ❌ Non-Causal (uses n+2)

🔒 BIBO Stability

Def:|x(t)| ≤ Mₓ → |y(t)| ≤ Mᵧ
LTI:∫|h(τ)|dτ < ∞

If bounded input produces bounded output → Stable

🌊 Fourier Analysis

Fourier Series · Fourier Transform · Properties · Pairs

📐 Fourier Series (Periodic)

Exp:x(t) = Σ Cₖ e^(jkω₀t)
Coeff:Cₙ = (1/T₀) ∫₀ᵀ x(t)e^(−jnω₀t)dt
Trig:x(t) = A₀ + Σ Aₖcos(kω₀t) + Bₖsin(kω₀t)
Even signal → Bₖ = 0 | Odd signal → Aₖ = 0, A₀ = 0
Aₖ:(2/T₀)∫x(t)cos(kω₀t)dt
Bₖ:(2/T₀)∫x(t)sin(kω₀t)dt

⚡ Fourier Transform (Aperiodic)

Forward:X(F) = ∫ x(t) e^(−j2πFt) dt
Inverse:x(t) = ∫ X(F) e^(+j2πFt) dF
Alt ω:X(ω) = ∫ x(t) e^(−jωt) dt
Parseval: ∫|x(t)|²dt = ∫|X(F)|²dF

🔺 Laplace Transform

Unilateral LT · Properties · Pairs · Inverse LT · Partial Fractions

📐 Definition

Bilateral:X(s) = ∫₋∞^∞ x(t)e^(−st)dt
Unilateral:X(s) = ∫₀^∞ x(t)e^(−st)dt
s:s = σ + jω = σ + j2πF
Inverse:x(t) = (1/2πj)∫ X(s)e^(st)ds
FT is special case of LT when s = j2πF (σ = 0, on jω axis)
ROC must include jω axis for FT to exist

🗺️ Pole-Zero & ROC

○Zeros: values of s where X(s) = 0
×Poles: values of s where X(s) = ∞
ROCRegion of Convergence — where LT exists
Causal → ROC: Re(s) > rightmost pole
Anti-causal → ROC: Re(s) < leftmost pole
Two-sided → ROC: strip between poles
Stable ↔ ROC includes jω axis

🧮 Partial Fraction Expansion

Form:X(s) = N(s)/D(s) = Σ Aₖ/(s−pₖ)
Residue:Aₖ = (s−pₖ)·X(s)|_{s=pₖ}
Repeated:X(s) = A₁/(s−p) + A₂/(s−p)² + ...
Steps: 1) Factor D(s) → find poles. 2) Write PFE form. 3) Compute residues Aₖ. 4) Inverse LT each term using tables.

✳️ Convolution

CT Convolution · DT Convolution · LTI Systems · Properties

📐 CT Convolution

Def:y(t) = x(t) ⊗ h(t)
Integral:y(t) = ∫₋∞^∞ x(τ)·h(t−τ)dτ
Alt:y(t) = ∫₋∞^∞ h(τ)·x(t−τ)dτ
Convolution in time → Multiplication in frequency!
Y(F) = X(F)·H(F)

🔢 DT Convolution

Sum:y[n] = x[n] ⊗ h[n]
Def:y[n] = Σₖ x[k]·h[n−k]
Alt:y[n] = Σₖ h[k]·x[n−k]
Length of output: if x has M points, h has N points → y has M+N−1 points

⚡ Convolution Properties

Commutative: x⊗h = h⊗x
Associative: (x⊗h₁)⊗h₂ = x⊗(h₁⊗h₂)
Distributive: x⊗(h₁+h₂) = x⊗h₁ + x⊗h₂
Identity: x(t)⊗δ(t) = x(t)

🏗️ LTI System & Impulse Response

h(t)Impulse response fully characterizes an LTI system
H(F)Transfer function = FT of h(t)
y(t)Output = input ⊗ impulse response
System is stable ↔ ∫|h(τ)|dτ < ∞
System is causal ↔ h(t) = 0 for t < 0

💻 MATLAB Code Library

Ready-to-use scripts for signal processing

01 · Sine Wave

t = 0:0.01:2*pi;
y = sin(t);
plot(t, y, 'b', 'LineWidth', 2);
grid on;
xlabel('t (s)'); ylabel('x(t)');
title('Sine Wave');

02 · Unit Step u(t)

t = -5:0.01:5;
u = double(t >= 0);
plot(t, u, 'r', 'LineWidth', 2);
ylim([-0.2 1.2]);
xlabel('t'); ylabel('u(t)');
title('Unit Step Function');

03 · Rectangular Pulse

t = -5:0.01:5;
A=2; t1=-1; t2=1;
y = A*(double(t>=t1) - double(t>=t2));
plot(t, y, 'g', 'LineWidth', 2);
title('Rectangular Pulse');

04 · Triangular Signal

t = -5:0.01:5;
A=2; T=2; t0=0;
y = A*max(1-abs((t-t0)/T), 0);
plot(t, y, 'm', 'LineWidth', 2);
title('Triangular Signal');

05 · Exp Decay

t = -1:0.01:5;
a = 1.5;
y = exp(-a*t) .* double(t>=0);
plot(t, y, 'k', 'LineWidth', 2);
title('Exponential Decay e^{-at}u(t)');

06 · Fourier Series Reconstruction

t = -pi:0.01:pi; N=10;
x = zeros(size(t));
for k = 1:2:2*N-1
  x = x + (4/pi)*(1/k)*sin(k*t);
end
plot(t, x, 'b', 'LineWidth', 2);
title('Square Wave FS (N=10 terms)');

07 · FFT (Frequency Spectrum)

fs = 1000; T = 1; t = 0:1/fs:T-1/fs;
x = cos(2*pi*50*t) + 0.5*sin(2*pi*120*t);
N = length(x);
X = fft(x)/N;
f = (0:N/2-1)*fs/N;
plot(f, 2*abs(X(1:N/2)));
xlabel('Frequency (Hz)');
title('Magnitude Spectrum |X(f)|');

08 · CT Convolution

dt = 0.01;
t = -2:dt:6;
x = double(t >= 0 & t <= 2); % rect
h = exp(-t) .* double(t >= 0); % decay
y = conv(x, h) * dt;
t_y = (0:length(y)-1)*dt + 2*t(1);
plot(t_y, y, 'r', 'LineWidth', 2);
title('Convolution y(t) = x(t)*h(t)');

09 · DT Convolution

x = [1 2 3 4]; % input signal
h = [1 1 1];   % impulse response
y = conv(x, h);
n = 0:length(y)-1;
stem(n, y, 'filled', 'LineWidth', 2);
xlabel('n'); ylabel('y[n]');
title('DT Convolution y[n] = x[n]*h[n]');

10 · Laplace (Symbolic)

syms t s
% Define signal
x = exp(-2*t) * heaviside(t);
% Compute Laplace Transform
X = laplace(x, t, s)
% Compute Inverse
x_back = ilaplace(X, s, t)
% Plot
fplot(x, [0 5]);
title('Signal and its LT');
