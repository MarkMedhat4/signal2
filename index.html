🔄 Even & Odd Decomposition

Even: xₑ(t) = [x(t) + x(−t)] / 2
Odd: xₒ(t) = [x(t) − x(−t)] / 2
Even × Even = Even
Odd × Odd = Even
Odd × Even = Odd
∫ Odd dt over [−T,T] = 0

CT:x(t+T₀) = x(t)

T₁/T₂ irrational → NOT Periodic

📏 Linearity Test

1Apply x₁ → get y₁, apply x₂ → get y₂
2Apply (ax₁ + bx₂) → get y₃
3Check: y₃ = ay₁ + by₂ ?
Disqualifiers (auto NL):
cos(·)   sin(·)   ln(·)   e^x   x²   |x|   +constant

⏱️ Time Invariance Test

1Shift input: x(t) → x(t−t₀) → compute y₁(t)
2Take output y(t) → replace t with (t−t₀) → y₂(t)
3If y₁ = y₂ → TIME INVARIANT ✅
TV Clues: coefficient depends on t (like eᵗ, t², sin(t)·x(t)), or input has time scaling x(at)

⏩ Causality

Causal: Output depends ONLY on present & past inputs

y[n] = x[n] + x[n−1] ✅ Causal

Non-Causal: Needs future inputs

y[n] = x[n+2] − x[n] ❌ Non-Causal (uses n+2)

🔒 BIBO Stability

Def:|x(t)| ≤ Mₓ → |y(t)| ≤ Mᵧ

If bounded input produces bounded output → Stable

✏️ Example 1 — Linearity & Time Shift

Problem: Find FT of g(t) = 0.5·g₁(t−2.5) + g₂(t−2.5)
where g₁(t) = rect(t/1) (width 1), g₂(t) = rect(t/3) (width 3), both centered at 0
1Rewrite: g(t) = 0.5·rect((t−2.5)/1) + 1·rect((t−2.5)/3)
2FT of rect(t/T) = T·sinc(ωT/2), so rect(t/1) → sinc(ω/2), rect(t/3) → 3·sinc(3ω/2)
3Apply Time Shift: f(t−t₀) → F(ω)·e^(−jωt₀), here t₀ = 2.5
4Result: G(f) = 0.5·sinc(f)·e^(−j5πf) + 3·sinc(3f)·e^(−j5πf)

✏️ Example 2 — Rect Pulse Direct Integration

Given: f(t) = A for 0 ≤ t ≤ τ, else 0
1F(ω) = ∫₀^τ A·e^(−jωt) dt
2= A·[e^(−jωt)/(−jω)]₀^τ = A·(e^(−jωτ)−1)/(−jω)
3Factor out e^(−jωτ/2): = 2Ae^(−jωτ/2)·sin(ωτ/2)/ω
4Result: F(ω) = e^(−jωτ/2)·Aτ·sinc(ωτ/2)

✏️ Example 3 — Time Scaling

Given: g(t) = 3·rect[(t−4)/2] → rewrite as 3·f(0.5t − 2)
1FT{rect(t)} = 2·sinc(ω)
2Time scaling a=0.5 → (1/0.5)·F(2ω) = 2·F(2ω)
3Time shift by t₀/a = 4: multiply by e^(−j4ω)
4Result: G(ω) = 6·sinc(ω)·e^(−j4ω)

✏️ Example 4 — FT of cos(ω₀t) and sin(ω₀t)

cos expand:cos(ω₀t) = ½·e^(jω₀t) + ½·e^(−jω₀t)
FT pair:e^(±jω₀t) ↔ 2πδ(ω ∓ ω₀)
→ FT cos:𝓕{cos(ω₀t)} = π[δ(ω−ω₀) + δ(ω+ω₀)]
sin expand:sin(ω₀t) = (1/2j)·e^(jω₀t) − (1/2j)·e^(−jω₀t)
→ FT sin:𝓕{sin(ω₀t)} = −jπ[δ(ω−ω₀) − δ(ω+ω₀)]

✏️ Example 5 — FT of Cosine Pulse (Windowed Cosine)

Given: f(t) = cos(ω₀t) for −τ < t ≤ τ, else 0
1Use cos = ½e^(jω₀t) + ½e^(−jω₀t), substitute into FT integral
2F(ω) = ½∫₋τ^τ e^(−j(ω−ω₀)t)dt + ½∫₋τ^τ e^(−j(ω+ω₀)t)dt
3Each integral evaluates to a sinc: I = τ·sinc[(ω−ω₀)τ]
4Result: F(ω) = τ·sinc[(ω−ω₀)τ] + τ·sinc[(ω+ω₀)τ]
Two sinc lobes centered at ±ω₀ — this is AM signal spectrum!

✏️ Example 6 — Frequency Shift (AM Modulation)

Given: g₃(t) = 10cos(200πt)cos(1000πt)
1Expand: g₃(t) = 5cos(200πt)e^(j1000πt) + 5cos(200πt)e^(−j1000πt)
2Apply frequency shift: multiply by e^(±jω₀t) → shift spectrum by ±ω₀
3G₃(ω) = 5π[δ(ω−1200π) + δ(ω−800π) + δ(ω+800π) + δ(ω+1200π)]
4IFT gives: g₃(t) = 5cos(1200πt) + 5cos(800πt)

✏️ Example 7 — FT of sgn(t)

sgn(t) = +1 for t>0, −1 for t<0
1Note: d/dt[sgn(t)] = 2δ(t)
2FT of 2δ(t) = 2
3Differentiation: jω·F(ω) = 2 → F(ω) = 2/jω
4Result: sgn(t) ↔ 2/jω

📝 Fourier Transform — Study Notes

Bilingual Reference · ω-notation · Core Ideas & Takeaways · مذكرة دراسية

💡 What is FT? | ما هو تحويل فورييه؟

The Fourier Transform converts a time-domain signal into its frequency-domain representation. Instead of describing how a signal changes over time, we describe what frequencies it contains and how strong each one is.

تحويل فورييه بيحوّل أي إشارة من نطاق الزمن (Time Domain) إلى نطاق التردد (Frequency Domain). بدل ما تقول الإشارة شكلها إيه مع الوقت، بتقول: مكوّنة من ترددات إيه وقد إيه قوة كل تردد.

📌
Core Idea | الفكرة الأساسية
Every signal = Sum of sine and cosine waves
أي إشارة = مجموع موجات جيبية (sin & cos)

✅ FT tells you: which frequencies exist + how strong each is
✅ FT بيقولك: أي ترددات موجودة + قد إيه كل تردد قوي

📐 FT Equations

Forward:X(ω) = ∫₋∞^∞ x(t) · e^(−jωt) dt
Inverse:x(t) = (1/2π) ∫₋∞^∞ X(ω) · e^(jωt) dω
x(t) → Signal in Time Domain
X(ω) → Signal in Frequency Domain
e^(−jωt) → Decomposes into sinusoids
ω → Angular frequency (rad/s)

🔑 Why Use FT? | ليه نستخدمه؟

Analysis in time domain → Complex. Analysis in frequency domain → Much easier! Analyzing audio, electrical signals, or waves becomes trivial once you know their frequency content.

بدل ما تحلل صوت، إشارة كهربائية، أو موجة في الزمن → روح تبص على الترددات اللي جواها. ده بيبسط الدنيا جداً!

⚡ Key Takeaways | الأفكار الرئيسية

⬌
Time ↔ Frequency

Every time signal has a frequency representation

كل إشارة زمنية ليها تمثيل تردد

✖️
Conv ↔ Multiply

Convolution in time = Multiplication in frequency

Convolution في الزمن = ضرب في التردد

📐
Shift → Phase

Time shift adds a phase factor, magnitude unchanged

الإزاحة في الزمن = Phase shift في التردد

🔄
Scaling Trade-off

Compress time → Expand frequency (and vice versa)

تضغط الزمن → تتوسع في التردد

⚡
δ(t) → All Freqs

Impulse contains ALL frequencies equally (flat spectrum)

النبضة فيها كل الترددات بالتساوي

🔲
Constant → DC Only

Constant signal has ONLY zero frequency

الإشارة الثابتة تردد صفر بس

🇦🇷 ملخص عربي — أهم نقاط FT

أهم الأزواج (لازم تحفظها):

  • e^(−at)u(t) → 1/(a + jω) ← حفظها! أكتر زوج بييجي في الامتحانات
  • δ(t) → 1 ← النبضة فيها كل الترددات
  • 1 → 2πδ(ω) ← الإشارة الثابتة تردد صفر بس
  • rect(t/τ) → τ·sinc(ωτ/2) ← مستطيل ↔ sinc

أهم الخواص (فهم مش حفظ):

  • الخطية: قسّم الإشارة الكبيرة لأجزاء أبسط
  • الإزاحة في الزمن: بتضيف phase shift في التردد
  • الـ Convolution: أهم خاصية — بتتحول لضرب بسيط
  • الـ Scaling: تضغط الزمن → توسع التردد والعكس

🎯 Practice Problems — Fourier Transform

📦 Problem Set 1 — Direct Fourier Transform | تحويل مباشر

Q1 Find the Fourier Transform of x(t) = e^(−3t)·u(t)

أوجد تحويل فورييه للإشارة x(t) = e^(−3t)·u(t)

▶ Show Solution | عرض الحل
1Apply the FT definition: X(ω) = ∫₀^∞ e^(−3t) · e^(−jωt) dt
2Combine exponentials: X(ω) = ∫₀^∞ e^(−(3 + jω)t) dt
3Evaluate: X(ω) = [ e^(−(3+jω)t) / (−(3+jω)) ]₀^∞ = 0 − (−1/(3+jω))
4X(ω) = 1 / (3 + jω)
X(ω) = 1 / (3 + jω)
Magnitude: |X(ω)| = 1/√(9 + ω²)
📌 Works when a > 0 (signal decays). General form: e^(−at)u(t) ↔ 1/(a+jω)
Q2 Find FT of x(t) = δ(t − 2)

أوجد تحويل فورييه لـ x(t) = δ(t − 2)

▶ Show Solution | عرض الحل
1FT definition: X(ω) = ∫₋∞^∞ δ(t − 2) · e^(−jωt) dt
2Sifting property picks out value at t = 2: X(ω) = e^(−jω·2)
X(ω) = e^(−j2ω)
📌 Magnitude: |X(ω)| = 1 (flat spectrum!)    Phase: ∠X(ω) = −2ω (linear phase)
Q3 Using properties: Find FT of x(t) = t · e^(−2t)·u(t)

باستخدام الخصائص: أوجد تحويل فورييه لـ x(t) = t · e^(−2t)·u(t)

▶ Show Solution | عرض الحل
1Known pair: e^(−2t)u(t) ↔ 1/(2 + jω)
2Use Freq Differentiation property: t·x(t) ↔ j · dX(ω)/dω
3Differentiate X(ω) = 1/(2 + jω): dX/dω = −j / (2 + jω)²
4Multiply by j: j · [−j/(2+jω)²] = 1/(2+jω)²
X(ω) = 1 / (2 + jω)²

📦 Problem Set 2 — Properties Application | تطبيق الخصائص

Q4 If X(ω) = 1/(1+jω), find FT of y(t) = x(t−3)·e^(j5t)

لو X(ω) = 1/(1+jω)، أوجد FT لـ y(t) = x(t−3)·e^(j5t)

▶ Show Solution | عرض الحل
1Apply Time Shift to x(t−3): → X(ω)·e^(−j3ω) = e^(−j3ω)/(1+jω)
2Apply Frequency Shift (multiplication by e^(j5t)): replace ω with (ω−5)
3Combine both operations
Y(ω) = e^(−j3(ω−5)) / (1 + j(ω−5))
= e^(−j3ω+j15) / (1 + j(ω−5))
Q5 Prove: FT{x(−t)} = X*(ω) for real x(t)

أثبت: FT{x(−t)} = X*(ω) للإشارات الحقيقية

▶ Show Solution | عرض الحل
1By definition: FT{x(−t)} = ∫₋∞^∞ x(−t)·e^(−jωt) dt
2Substitute τ = −t → dτ = −dt, limits flip: = ∫₋∞^∞ x(τ)·e^(jωτ) dτ
3This is X(ω) evaluated at −ω: = X(−ω)
4For real x(t), conjugate symmetry applies: X(−ω) = X*(ω) ∎
FT{x(−t)} = X(−ω) = X*(ω)

📦 Problem Set 3 — System Analysis | تحليل السيستمز

Q6 LTI system: h(t) = e^(−t)·u(t), input x(t) = e^(−2t)·u(t). Find y(t).

سيستم LTI عنده h(t) = e^(−t)u(t) والمدخل x(t) = e^(−2t)u(t). أوجد y(t).

▶ Show Solution | عرض الحل
1Find transforms: H(ω) = 1/(1+jω)  and  X(ω) = 1/(2+jω)
2Y(ω) = X(ω)·H(ω) = 1/[(1+jω)(2+jω)]
3Partial Fractions: Y(ω) = A/(1+jω) + B/(2+jω)
At jω=−1: A = 1/(2−1) = 1  |  At jω=−2: B = 1/(1−2) = −1
4Inverse FT each term
y(t) = [e^(−t) − e^(−2t)] · u(t)
📌 Convolution in time → Multiplication in frequency → Partial fractions → Inverse FT

🔺 Laplace Transform

Unilateral LT · Properties · Pairs · Inverse LT · Partial Fractions

📐 Definition

Bilateral:X(s) = ∫₋∞^∞ x(t)e^(−st)dt

🗺️ Pole-Zero & ROC

○Zeros: values of s where X(s) = 0
×Poles: values of s where X(s) = ∞
ROCRegion of Convergence — where LT exists
Causal → ROC: Re(s) > rightmost pole
Anti-causal → ROC: Re(s) < leftmost pole
Two-sided → ROC: strip between poles
Stable ↔ ROC includes jω axis

🧮 Partial Fraction Expansion

Form:X(s) = N(s)/D(s) = Σ Aₖ/(s−pₖ)

Repeated:X(s) = A₁/(s−p) + A₂/(s−p)² + ...
Steps: 1) Factor D(s) → find poles. 2) Write PFE form. 3) Compute residues Aₖ. 4) Inverse LT each term using tables.

✳️ Convolution

CT Convolution · DT Convolution · LTI Systems · Properties

📐 CT Convolution

Def:y(t) = x(t) ⊗ h(t)
Integral:y(t) = ∫₋∞^∞ x(τ)·h(t−τ)dτ
Alt:y(t) = ∫₋∞^∞ h(τ)·x(t−τ)dτ
Convolution in time → Multiplication in frequency!
Y(F) = X(F)·H(F)

🔢 DT Convolution

Sum:y[n] = x[n] ⊗ h[n]
Def:y[n] = Σₖ x[k]·h[n−k]
Alt:y[n] = Σₖ h[k]·x[n−k]
Length: if x has M points, h has N points → y has M+N−1 points

⚡ Properties

Commutative: x⊗h = h⊗x
Associative: (x⊗h₁)⊗h₂ = x⊗(h₁⊗h₂)
Distributive: x⊗(h₁+h₂) = x⊗h₁ + x⊗h₂
Identity: x(t)⊗δ(t) = x(t)

🏗️ LTI System & Impulse Response

h(t)Impulse response fully characterizes an LTI system
H(F)Transfer function = FT of h(t)
y(t)Output = input ⊗ impulse response
System stable ↔ ∫|h(τ)|dτ < ∞
System causal ↔ h(t) = 0 for t < 0

06 · Fourier Series Reconstruction

t = -pi:0.01:pi; N=10;

for k = 1:2:2*N-1
  x = x + (4/pi)*(1/k)*sin(k*t);
end
plot(t, x, 'b', 'LineWidth', 2);
title('Square Wave FS (N=10 terms)');

07 · FFT (Frequency Spectrum)

fs = 1000; T = 1; t = 0:1/fs:T-1/fs;
x = cos(2*pi*50*t) + 0.5*sin(2*pi*120*t);
N = length(x);
X = fft(x)/N;
f = (0:N/2-1)*fs/N;
plot(f, 2*abs(X(1:N/2)));
xlabel('Frequency (Hz)');
title('Magnitude Spectrum |X(f)|');

08 · CT Convolution

dt = 0.01;
t = -2:dt:6;

h = exp(-t) .* double(t >= 0);
y = conv(x, h) * dt;
t_y = (0:length(y)-1)*dt + 2*t(1);
plot(t_y, y, 'r', 'LineWidth', 2);
title('Convolution y(t) = x(t)*h(t)');
